# MultiFlash Tool

一款开源的多功能安卓刷机工具，支持高通 EDL (9008) 模式和 Fastboot 模式。

## 功能特性

- 📱 **设备检测** - 自动识别 ADB/Fastboot/EDL 设备
- 🔧 **EDL 刷写** - 支持高通 9008 模式刷机
- ⚡ **Fastboot 操作** - 分区读写、OEM 解锁
- 📦 **Payload 提取** - 从 OTA 包提取分区镜像
- 🔄 **Super 合并** - 动态分区合并工具

## 系统要求

- Windows 10/11
- .NET Framework 4.8
- 高通驱动 (EDL 模式需要)

## 快速开始

1. 下载最新 Release
2. 解压到任意目录
3. 运行 `MultiFlash Tool.exe`
4. 连接设备并选择操作模式
